STS Card Art Speedrunning

A tool to help you draw a whole set of card art for your modded Slay the Spire character quickly.
Set a short time limit, like one minute per card, input your card data, then go ham!
When you're finished, a ZIP will be provided with all the art (including the scaled-up portrait versions), cropped to the frame.
Also useful for beta art.

NOTE: this currently only works on desktop browsers.

Paste your CardStrings.json here then click Load.
Leave this blank if you'd rather enter card data manually.
